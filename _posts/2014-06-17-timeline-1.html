rationale

the goal of this project is to create a state-of-art responsive story replacing csongor's current portfolio.

there are many web designers and developers out there, a simple portfolio is worth almost nothing. A good storytelling incorporating the latest design & development trends might be unique in the industry.

approach

with this project we will show how to create beautiful stories on the web for all devices.

we will look for beauty which in our case represents the harmony between oneness / wholeness and diversity.

project plan

we will create a couple of slides / single page sub-stories for the following concepts:

  1. project title, short description
  2. typographic grid
  3. classic, asymmetric, centered grid
  4. images
  5. common web components
  6. about the author

deliverables

  1. this story as a portfolio replacement
  2. a styleguide associated
  3. a set of re-usable web components with full documentation
